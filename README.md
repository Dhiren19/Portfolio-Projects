## Repository Structure

This repository contains multiple data analytics projects.  
Each project is organized in its own folder as listed below:

- **excel-supermarket-dashboard** – Interactive Excel sales dashboard  
- **powerbi-dashboard-project-1 (IBM HR Attrition Analytics)** – Two-page Power BI sales dashboard  
- **powerbi-dashboard-project-2 (Udemy Finance & Accounting Course)** – Professionally styled Power BI dashboard  
- **sql-airbnb-analysis** – SQL-based Airbnb analytics project
